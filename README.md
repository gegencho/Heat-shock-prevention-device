# Heat-shock protection system

親機用ソフト(Heatshock_P)と子機用ソフト(Heatshock_C)から成り立つ

##### [制作者]gegencho

##### [動作環境]

	Python  
	温湿度センサーDHT22  
	入力ボタン  
	RGB LED  
	アクティブブザー
